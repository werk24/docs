The following file formats are supported.

| **Extension** | **Name**                  |
|---------------|---------------------------|
| .bmp          | Windows bitmaps           |
| .jp2          | JPEG 2000 files           |
| .jpeg         | JPEG files                |
| .png          | Portable Network Graphics |
| .tiff         | TIFF files                |
| .pdf          | Portable Document Format (Vectorized and Raster) |
